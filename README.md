

 Watchlist de Filmes

Aplicação web simples de lista de filmes com login e CRUD completo.

## Como usar

1. Abra o arquivo index.html no navegador
2. Faça login com:
   - Usuário: aluno
   - Senha: fiap2025
3. Adicione, edite e remova filmes da lista

## Arquivos

- index.html
- style.css
- script.js
- prompts.md
